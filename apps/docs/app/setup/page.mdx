## Setup an AI Inference Provider

AI Agents use Inference to generate responses. This process requires access to a large language model, or access to such a model via an API. For our workshop we are going to leverage hosted inference providers.

    - **Note:** You'll need to update the model in `apps/agent/app/api/chat/route.ts` to use `model: anthropic('claude-3-haiku-20240307')`
  </TabsContent>

    - **Create account:** <ExternalLink href="https://vercel.com/signup">vercel.com/signup</ExternalLink>

    - **Note:** You'll need to update the model in `apps/agent/app/api/chat/route.ts` to use `model: 'openai/gpt-oss-120b'` or your preferred model
  </TabsContent>
</Tabs>

You'll need an API key from one of these providers. The starter-kit uses <ExternalLink href="https://ai-sdk.dev">Vercel AI SDK</ExternalLink> to abstract the inference providers.


## Clone the workshop repository

First lets clone the git repository, so that we can get template and all the utilities for the workshop

```bash 
git clone https://github.com/ciamshrek/demo-trade-pro.git
cd demo-trade-pro
nvm use --lts           # if using nvm
pnpm install      # install dependencies
```

## Start your template agent

Now let's add your AI API key and run the agent:

<Tabs defaultValue="openai" className="w-full">
  <TabsList className="grid w-full grid-cols-3">
    <TabsTrigger value="openai">OpenAI</TabsTrigger>
    <TabsTrigger value="anthropic">Anthropic</TabsTrigger>
    <TabsTrigger value="vercel">Vercel AI Gateway</TabsTrigger>
  </TabsList>
  
  <TabsContent value="openai">
    ```bash
    cd apps/agent
    
    # Create .env.local file with your OpenAI API key
    nano .env.local  # or code .env.local, vim .env.local, etc.
    
    # Add this line to the file:
    # OPENAI_API_KEY=your_actual_openai_key_here
    
    # Save and exit, then start the agent
    pnpm dev  # starts on port 3003
    ```
  </TabsContent>
  
  <TabsContent value="anthropic">
    ```bash
    cd apps/agent
    
    # Create .env.local file with your Anthropic API key
    nano .env.local  # or code .env.local, vim .env.local, etc.
    
    # Add this line to the file:
    # ANTHROPIC_API_KEY=your_actual_anthropic_key_here
    
    # Save and exit, then start the agent
    ```
